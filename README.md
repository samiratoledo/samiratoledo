### 📂 Projetos em Destaque

- **[ClimaX](https://github.com/samiratoledo/climax)** — Aplicação web de previsão climática em tempo real, com interface Glassmorphism, temas dinâmicos e consumo da API OpenWeather via Fetch/Async-Await. Implementa boas práticas de segurança, mantendo a chave de API fora do repositório via `.gitignore`.
- **[MyPocket](https://github.com/samiratoledo/mypocket)** — Ecossistema web para controle de endividamento e saúde financeira doméstica, com front-end responsivo (HTML, CSS, Bootstrap) e lógica de back-end em PHP e JavaScript.
- **[JavaLog Express](https://github.com/samiratoledo/javalogexpress)** — Sistema em Java para controle e registro de veículos, com cálculo automatizado de fretes para uma empresa fictícia de logística.
- **[Bibliotech](https://github.com/samiratoledo/biblioteca-poo)** — Aplicação em Java, via terminal, para gerenciamento de acervo de biblioteca, aplicando conceitos de Programação Orientada a Objetos.
- **[Revista ZL](https://www.figma.com/design/fCCutjwfeCywoYxOcrmzlt/Revista-Zona-Jovem)** — Revista digital de 98 páginas, com liderança na coordenação de equipes de criação de conteúdo. Design e diagramação em Figma.
